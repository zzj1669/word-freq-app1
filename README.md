# 中文网页词频分析与可视化系统

基于 Streamlit + pyecharts 的中文网页文本词频分析与可视化系统，支持多种图表展示方式。

## 功能特性

- **网页抓取**：支持从任意中文网页提取正文内容
- **中文分词**：基于 jieba 进行精准中文分词
- **词频统计**：自动统计高频词汇，支持自定义过滤阈值
- **多种可视化**：提供 7 种图表展示方式
  - 词云图：直观展示词汇权重
  - 柱状图：清晰对比词频差异
  - 折线图：展示词频趋势
  - 饼图：显示词频占比
  - 雷达图：多维度词汇分析
  - 散点图：词汇分布展示
  - 漏斗图：词频层级展示
- **数据导出**：支持将词频数据导出为 CSV 文件
- **交互式界面**：提供侧边栏配置，支持动态调整参数

## 使用方法

### 启动应用

```bash
streamlit run app.py
```

启动后，浏览器将自动打开应用页面（默认地址：`http://localhost:8501`）。

### 操作步骤

1. **输入网页 URL**：在主界面输入框中输入任意中文网页地址
2. **配置图表参数**（可选）：
   - 在侧边栏选择图表类型
   - 设置最低词频过滤阈值
   - 调整显示词汇数量（Top-N）
3. **查看结果**：
   - 页面将显示网页标题和文本统计信息
   - 展示词频排名表格
   - 显示选中类型的可视化图表
4. **导出数据**：点击「下载词频CSV」按钮导出数据

## 配置说明

### 停用词表

停用词表位于 `stopwords_cn.txt`，每行一个停用词。系统会自动过滤这些无意义词汇。如需扩展或修改，直接编辑该文件即可。

### 图表参数

在应用侧边栏可配置以下参数：

- **图表类型**：选择 7 种可视化方式之一
- **最低词频过滤**：过滤词频低于设定值的词汇（1-20）
- **显示词汇数量**：设置图表中显示的高频词汇数量上限（5-50）

## 常见问题

### Q1：抓取网页失败？

- 检查网络连接是否正常
- 确认 URL 格式正确（需包含 `http://` 或 `https://`）
- 部分网站可能有反爬机制，可尝试更换其他网页

### Q2：词频统计结果不准确？

- 停用词表可能不够完善，可手动扩展 `stopwords_cn.txt`
- 可调整「最低词频过滤」参数过滤低频无意义词

### Q3：图表显示异常？

- 确保所有依赖已正确安装
- 检查网络是否能正常加载 ECharts 资源
- 尝试刷新页面或重新启动应用
